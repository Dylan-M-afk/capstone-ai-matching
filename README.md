# AI-Assisted Student-Employer Matching Platform

A Web platform designed to connect college students with local companies using an artificial intelligence system (LLM) which allows applicants to be intelligently ranked for job vacancies. 

>! Disclaimer: This is NOT a fill commercial product. It is a MVP of the proposed system

## Key Features

Students
- Register accounts with college email
- Create & edit their profile
- Upload CV for AI extracted profile data
- View and apply to existing job vacancies

Companies
- login with admin-provided credentials
- post job vacancies
- view applicants for existing vacancies
- generate AI rankings
- view comprehensive scores and explanations

Administrators
- Create company accounts
- generate company access codes
- view all students & Companies
- manage accounts 
- administrator dashboard
